# NoteAnchor Landing Page Draft

## Headline

Select a phrase. Add a note. Keep the context.

## Short intro

NoteAnchor is a local desktop app for active reading and contextual notes.

It lets you select an exact word, phrase, or sentence in a document and attach a note to that exact passage. The note stays connected to the original text, so you can return later and still see exactly what it refers to.

The original document is not changed. NoteAnchor keeps note data beside the source file instead of writing comments into the document itself.

## The problem

Notes lose value when they are separated from the exact text they refer to.

A comment like "too vague", "check this source", "better option here", or "important phrase" only helps when it remains attached to the specific passage that triggered it. Once notes are moved into a separate file, notebook, or memory, much of that precision disappears.

## What NoteAnchor does

NoteAnchor helps you read closely without changing the source document.

* Open TXT documents.
* Open DOCX files as plain text.
* Attach notes to selected words, phrases, or sentences.
* Review all notes in one place.
* Return to the original passages later.
* Reconnect a note if the source text changes.
* Export notes.
* Save a print-ready report.

## Who it may be useful for

NoteAnchor may be useful for:

* editors and proofreaders reviewing drafts;
* researchers and analysts collecting observations from source texts;
* writers and authors working through drafts or reference material;
* literary translators marking difficult phrases, stylistic choices, and possible alternatives;
* people learning to read in a new language.

For people learning to read in a new language, NoteAnchor can serve as a vocabulary notebook with context: collect useful words and expressions together with the original sentence while reading real texts.

## Local-first

NoteAnchor runs on your computer. Your files stay on your computer, original documents are not modified, and notes are stored beside the document.

## Current beta limitations

* DOCX opens as plain text.
* Document formatting is not preserved.
* Print report saves a print-ready report that can be opened in a browser.

## Product principles

* Usefulness first.
* No exaggerated promises.
* Describe only what the app actually does.
* Keep original documents unchanged.
