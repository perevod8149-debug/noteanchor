Привет!

Хочу попросить тебя немного потестировать раннюю бета-версию моей программы NoteAnchor для Windows.

Это локальное приложение для внимательного чтения и контекстных заметок. В нем можно открыть текст, выделить отдельное слово, фразу или предложение и прикрепить заметку именно к этому фрагменту. Важный момент: исходный документ приложение не изменяет.

Почему я прошу об этом: сейчас мне важнее не похвала, а понять, насколько вообще понятны установка, запуск и базовое использование. Это ранняя бета, поэтому честная обратная связь для меня полезнее, чем просто "все хорошо".

Если будет время, пожалуйста, попробуй:

- установить и запустить приложение;
- открыть TXT-файл;
- открыть DOCX-файл как обычный текст, без форматирования Word;
- выделить слово, фразу или предложение;
- добавить заметку;
- открыть All notes, то есть список всех заметок;
- попробовать Find, то есть поиск по тексту;
- попробовать Whole word Find, то есть поиск только целых слов;
- сделать Export notes, то есть сохранить заметки в отдельный файл;
- сохранить print-ready report, который при необходимости потом можно открыть в браузере;
- если получится, закрыть и заново открыть приложение или документ.

Чего пока не стоит ожидать:

- форматирование DOCX не сохраняется;
- изображения, headers / верхние колонтитулы, footers / нижние колонтитулы, comments / комментарии, tracked changes / режим исправлений и сложная верстка в текущем DOCX-режиме не поддерживаются;
- RTF, PDF и документы, сделанные из сканов или изображений, пока не поддерживаются;
- Print report не печатает напрямую из NoteAnchor, а сохраняет файл, который потом можно открыть в браузере и распечатать уже оттуда.

Для безопасности лучше тестировать на копиях или на неважных документах. По идее приложение не должно изменять исходные TXT или DOCX-файлы. Заметки сохраняются рядом с документом в отдельных файлах NoteAnchor.

Что мне особенно важно узнать:

- сработала ли установка;
- было ли что-то непонятно;
- на каком шаге ты застрял;
- было ли действие, которое не сработало;
- какой тип документа ты тестировал;
- какая у тебя версия Windows, если знаешь;
- видел ли ты странные символы, сломанный текст или проблемы с кодировкой.
