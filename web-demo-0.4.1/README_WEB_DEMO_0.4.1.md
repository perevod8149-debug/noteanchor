# NoteAnchor 0.4.1 Web Demo Mockup

This folder contains a small local webpage that visually demonstrates the NoteAnchor workflow in motion.

It is a mockup, not the real NoteAnchor app.

## Files included

- `index.html`
- `styles.css`
- `demo.js`
- `README_WEB_DEMO_0.4.1.md`

## How to open it locally

Open `index.html` in a browser.

You can double-click the file in Windows Explorer, or drag it into a browser window.

No build step is required.
No dev server is required.
No external dependencies are used.

## What it shows

The page animates a simplified NoteAnchor workflow:

1. Open a sample document.
2. Select an exact phrase.
3. Add a note.
4. Show the note card beside the text.
5. Open All notes and search.
6. Show a translation-style note example.
7. Show Print report.
8. End with the local/privacy message.

The animation can be replayed with the `Replay demo` button.

## Important note

This is only a visual demo page for explanation and early presentation.

It does not use the real Tauri app, real note storage, or real document handling.

## Project safety

No app code was changed to create this demo.
No package or version files were changed.
