# NoteAnchor 0.4.1 Demo Recording Script

## Goal of the video

Show the core workflow quickly:

open a document -> select text -> add note -> see note beside text -> find notes in All notes -> create Print report.

## Recording setup

- Use the Windows desktop app.
- Use sample files from `tester-demo-0.4.1`.
- Do not show personal documents.
- Set window size around 1280x720 if possible.
- Increase app zoom only if text is too small.
- Hide unnecessary desktop clutter.
- Keep recording under 75 seconds.
- Record as MP4 first.

## Before recording

- Launch NoteAnchor.
- Make sure no personal file is open.
- Have `tester-demo-0.4.1` folder ready.
- Optional: clear old sample notes for the sample files if needed.
- Have exact note texts ready to type or paste.

## Timeline storyboard

### Scene 1: 0-5 seconds - Open NoteAnchor

- Show clean NoteAnchor window.
- Optional caption: `Local contextual notes for text documents.`

### Scene 2: 5-15 seconds - Open sample-review-text.txt

- Click Open `.txt`.
- Choose `sample-review-text.txt`.
- Show document loaded.

### Scene 3: 15-28 seconds - Add an editing note

- Select the phrase `modest boldness`.
- Click `Add note`.
- Type or paste:
  `Too vague - rewrite this phrase.`
- Save note.
- Show note card beside the text.

### Scene 4: 28-40 seconds - Add a second note / show compact note behavior

- Select phrase `timeless new character`.
- Add note:
  `Sounds polished but unclear - needs evidence.`
- Save.
- Click between notes if useful.
- Show one active note and another compact inactive note.

### Scene 5: 40-52 seconds - All notes

- Click `All notes`.
- Search for `vague` or `evidence`.
- Show filtered result.
- Click Open on the result.
- Show app returning to the selected note.

### Scene 6: 52-66 seconds - Translation-style note

- Open `sample-translation-text.txt`.
- Select one sentence or short paragraph from the first paragraph.
- Add note:
  `Translation draft: Після дощу вулиця не одразу посвітлішала.`
- Save.
- Show note beside paragraph.

### Scene 7: 66-75 seconds - Print report / closing privacy message

- Click `Print report`.
- Show readable report briefly.
- Closing caption:
  `Documents stay on your computer. Original files are not changed.`

## Exact text snippets to use

- Selected phrase 1: `modest boldness`
- Note 1: `Too vague - rewrite this phrase.`
- Selected phrase 2: `timeless new character`
- Note 2: `Sounds polished but unclear - needs evidence.`
- Search term: `vague` or `evidence`
- Translation note text:
  `Translation draft: Після дощу вулиця не одразу посвітлішала.`

## What to avoid

- Do not show Windows warnings in the demo video.
- Do not show Google Drive download process.
- Do not show personal folders if possible.
- Do not spend too long in file picker.
- Do not type long text slowly; paste prepared text if needed.
- Do not explain every feature.
- Do not show failed actions.

## Optional captions for final video

- `Open a TXT or DOCX document`
- `Attach a note to an exact phrase`
- `Find all notes in one panel`
- `Export or print a readable report`
- `Local app. Original documents stay unchanged.`

## Success criteria

The demo is successful if a viewer understands in under 75 seconds:

- what NoteAnchor does;
- how a note is attached;
- how notes are found again;
- why local/private document handling matters.
