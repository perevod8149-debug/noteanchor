2. Чи було зрозуміло, для чого потрібен NoteAnchor?

3. Який сценарій виявився для тебе найкориснішим: редагування, переклад, читання/дослідження чи жоден?

4. Що було незрозумілим?

5. Що не спрацювало?

6. Чи було легко знову знаходити нотатки?

7. Чи був корисним `All notes`?

8. Чи був корисним `Print report`?

9. Чи використовував(ла) б ти це для реального документа? Якщо так, то для чого?

10. Що одне варто покращити насамперед?
