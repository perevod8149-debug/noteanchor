# NoteAnchor 0.4.1 Guided Test

This is a guided test of NoteAnchor 0.4.1.

The goal is not to praise the app. The goal is to notice what feels clear, useful, confusing, awkward, or broken.

Please test on copies or on the sample files in this folder.

Original documents should not be modified by NoteAnchor.

Notes are saved beside the document as `.notes.json`.

## Scenario 1 - Review / editing notes

1. Open `sample-review-text.txt`.
2. Select one word or a short phrase.
3. Click `Add note`.
4. Write a note such as `unclear`, `rewrite`, or `check this`.
5. Open `All notes`.
6. Use `Search`.
7. Open a note from the list.

## Scenario 2 - Translation-style notes

1. Open `sample-translation-text.txt`.
2. Select a full sentence or paragraph.
3. Click `Add note`.
4. Write a translation or paraphrase as the note.
5. Add 2-3 such notes.
6. Check that inactive note cards stay compact.
7. Open `Print report` and check whether the original text and notes are readable.

## Scenario 3 - Research / reading notes

1. Open either sample file.
2. Select an important sentence.
3. Add a note such as `important idea`, `quote later`, or `check source`.
4. Use `All notes` to find it again.
