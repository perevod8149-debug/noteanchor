# NoteAnchor 0.4.1 - тест по шагам

Это тест по шагам для NoteAnchor 0.4.1.

Цель не в том, чтобы хвалить программу. Цель в том, чтобы заметить, что кажется понятным, полезным, неудобным, странным или сломанным.

Пожалуйста, тестируй на копиях файлов или на примерах из этой папки.

NoteAnchor не должен изменять оригинальные документы.

Заметки сохраняются рядом с документом в файле `.notes.json`.

## Сценарий 1 - Заметки для редактирования

1. Открой `sample-review-text.txt`.
2. Выдели одно слово или короткую фразу.
3. Нажми `Add note`.
4. Напиши заметку, например `unclear`, `rewrite` или `check this`.
5. Открой `All notes`.
6. Используй `Search`.
7. Открой одну заметку из списка.

## Сценарий 2 - Заметки для перевода

1. Открой `sample-translation-text.txt`.
2. Выдели полное предложение или абзац.
3. Нажми `Add note`.
4. Напиши в заметке перевод или перефразирование.
5. Добавь 2-3 такие заметки.
6. Проверь, что неактивные карточки заметок остаются компактными.
7. Открой `Print report` и посмотри, удобно ли читать исходный текст и заметки.

## Сценарий 3 - Заметки для чтения или исследования

1. Открой любой из двух файлов-примеров.
2. Выдели важное предложение.
3. Добавь заметку, например `important idea`, `quote later` или `check source`.
4. Используй `All notes`, чтобы найти её снова.
