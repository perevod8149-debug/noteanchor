# NoteAnchor 0.4.1 - тестування за інструкцією

Це тестування NoteAnchor 0.4.1 за інструкцією.

Мета не в тому, щоб хвалити програму. Мета в тому, щоб помітити, що виглядає зрозумілим, корисним, незручним, дивним або просто не працює.

Будь ласка, тестуй на копіях файлів або на прикладах із цієї папки.

NoteAnchor не має змінювати оригінальні документи.

Нотатки зберігаються поруч із документом у файлі `.notes.json`.

## Сценарій 1 - Нотатки для редагування

1. Відкрий `sample-review-text.txt`.
2. Виділи одне слово або коротку фразу.
3. Натисни `Add note`.
4. Напиши нотатку, наприклад `unclear`, `rewrite` або `check this`.
5. Відкрий `All notes`.
6. Скористайся `Search`.
7. Відкрий одну нотатку зі списку.

## Сценарій 2 - Нотатки для перекладу

1. Відкрий `sample-translation-text.txt`.
2. Виділи повне речення або абзац.
3. Натисни `Add note`.
4. Напиши в нотатці переклад або перефразування.
5. Додай 2-3 такі нотатки.
6. Перевір, що неактивні картки нотаток залишаються компактними.
7. Відкрий `Print report` і подивися, чи зручно читати оригінальний текст і нотатки.

## Сценарій 3 - Нотатки для читання або дослідження

1. Відкрий будь-який із двох файлів-прикладів.
2. Виділи важливе речення.
3. Додай нотатку, наприклад `important idea`, `quote later` або `check source`.
4. Скористайся `All notes`, щоб знайти її знову.
