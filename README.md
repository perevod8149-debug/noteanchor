# NoteAnchor

NoteAnchor is a local Windows app for adding notes to documents while keeping the original file unchanged.

It is built for reading and review work where notes need to stay connected to the passage, page, or document that triggered them.

## What NoteAnchor does

- Select text and attach a note to that passage
- Add point notes to visible locations on a page
- Add page notes for broader comments
- Keep notes separate from the source document
- Work locally on your own computer
- Avoid forced cloud upload

## Supported formats in 0.4.1

- TXT
- DOCX opened as plain text
- guarded PDF workflow

This is an early Windows release.

- the current release is not code-signed yet
- SmartScreen help is available on the website

## Current limitations

- PDF support is guarded and depends on document layout
- Some PDFs remain limited or preview-only
- DOCX content is currently handled as plain text
- DOCX formatting is not preserved
- PDF support is not complete across all documents

## Local-first workflow

- Files stay on your computer
- Notes are saved separately
- The original file is not changed
- The workflow does not require forced cloud upload
