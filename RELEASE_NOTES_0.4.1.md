# NoteAnchor 0.4.1 Release Notes

## What changed since 0.4.0

- Inactive note cards are more compact:

  - long selected/commented text is shortened in inactive cards;
  - long note/comment text is shortened in inactive cards;
  - active cards still show full content.

- Print report is cleaner:

  - long selected fragments and full paragraphs are no longer repeated twice;
  - one-word and short-phrase notes still keep useful sentence/phrase context.

- All notes panel is easier to use with many notes:

  - the header/search area now stays visible;
  - only the notes list scrolls;
  - the header was made more compact.

## Unchanged

- TXT support unchanged.
- DOCX plain-text support unchanged.
- Notes storage unchanged.
- Existing `.notes.json` files remain compatible.
- Export notes unchanged.
- Help unchanged.
- No PDF/RTF support yet.
- No cross-paragraph selection yet.
- No Word comments/tracked changes support yet.
