# NoteAnchor 0.4.1

## Release summary

Version 0.4.1 is the first public Windows release of NoteAnchor.

NoteAnchor is a local Windows app for adding contextual notes to documents without changing the original file.

## Included in 0.4.1

- Text notes
- Point notes
- Page notes
- TXT workflow
- DOCX workflow
- guarded PDF workflow

## What users can do

- Open local documents
- Add notes that stay tied to the relevant text, page, or page location
- Reopen documents and continue working with saved notes
- Use the current release without forced cloud upload

## Supported formats

- TXT
- DOCX
- PDF

## Known limitations

- PDF support is guarded and depends on the document layout
- Some PDFs support text notes well, while some remain limited or preview-only
- DOCX content is currently handled as plain text
- Windows SmartScreen may appear because this early release is not code-signed yet

## Release note

This release is intentionally small in scope. It focuses on a working local note workflow rather than a broad feature set.
